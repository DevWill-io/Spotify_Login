# 🎵 Spotify Login Clone

Uma recriação visual da página de login/cadastro do Spotify, focada em design limpo, responsividade e fidelidade à marca. Este projeto foi desenvolvido para praticar estruturação semântica com **HTML5** e estilização avançada com **CSS3**.

---

## 🚀 Tecnologias Utilizadas

* **HTML5:** Tags semânticas para melhor acessibilidade.
* **CSS3:** Flexbox para centralização, variáveis de cores e efeitos de transição.
* **Font Awesome:** Biblioteca de ícones para as redes sociais e o logotipo.
* **Google Fonts:** Tipografia baseada na identidade visual clássica.

---

## 🎨 Funcionalidades

* **Design Dark Mode:** Seguindo a paleta de cores oficial (#121212).
* **Botões Interativos:** Efeitos de `hover` e escalas suaves ao interagir.
* **Layout Responsivo:** O container se adapta a diferentes larguras de tela, mantendo o foco no centro.
* **Divisores Customizados:** Uso de pseudo-elementos (`::before` e `::after`) para criar a linha separadora "ou".
